# 🔍 Search History for VS Code

A lightweight Visual Studio Code extension that adds **search history tracking + sidebar panel**, bringing a workflow similar to MPLAB and other IDEs.

---

## ✨ Features

✅ **Search History Tracking**
- Automatically store every global search (`Ctrl + Shift + F`)

✅ **Sidebar Panel**
- Dedicated panel in the Activity Bar
- Quickly view previous searches

✅ **One‑Click Re-search**
- Click any item → instantly run the search again

✅ **Smart Search Behavior**
- Uses selected text automatically
- Falls back to input box if nothing selected

✅ **Right‑click Actions**
- Delete individual entries
- Clear entire history

✅ **Keyboard Shortcuts**
- `Ctrl + Shift + F` → Search + save history
- `Ctrl + Shift + H` → Open search history panel

---

## 🚀 Usage

### 🔍 Perform a Search

- Select text and press:
  ```
  Ctrl + Shift + F
  ```
  → Automatically searches selection

- Or press:
  ```
  Ctrl + Shift + F
  ```
  and enter your query manually

---

### 📂 Open Search History Panel

Press:

```
Ctrl + Shift + H
```

Or click the **Search History icon** in the left sidebar.

---

### 🔁 Re-run a Search

- Open the panel
- Click any previous search

---

### 🧹 Manage History

Right-click any item:

- **Delete** → removes a single search

Top bar of panel:

- **Clear All** → removes all history

---

## 🧠 Motivation

VS Code does not provide a visual way to manage past searches like some IDEs (e.g., MPLAB).

This extension adds:

- A **persistent history**
- A **visual list**
- A much faster navigation workflow

---

## ⚙️ Requirements

- VS Code 1.80 or newer
